## English

This repository is dedicated to the development of the **Lightning Bolt Network**, a next-generation internet architecture designed to solve the core vulnerabilities of the current centralized web. Our vision is to create a **Unified, Self-Sovereign, and Resilient Information Fabric**.

The architecture is built upon three interconnected pillars:

| Pillar | Concept | Goal |
| :--- | :--- | :--- |
| **1. Identity** | **Self-Sovereign Identity (SSI)** | Users own their identity and data via cryptographic keys (DIDs & VCs). |
| **2. Data** | **Content-Based Addressing (CBA)** | Data is addressed by its content hash (CID), making it immutable and censorship-resistant (e.g., IPFS). |
| **3. Infrastructure** | **Decentralized Physical Infrastructure Networks (DePIN)** | Community-owned, token-incentivized networks for compute, storage, and connectivity. |

### Phi Chain as the Nexus

The **Phi Chain** serves as the core trust and coordination layer (the Nexus) for this entire architecture. It provides the necessary Distributed Ledger Technology (DLT) to anchor DIDs, coordinate DePIN incentives, and facilitate secure data transactions across the network.
